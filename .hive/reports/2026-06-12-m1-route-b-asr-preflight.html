Composer captions 格式

manifest.audio.captions 是一个 JSON 路径；compose.mjs 读取该文件的顶层 captions 数组。

{
  "captions": [
    {
      "start": 0.32,
      "end": 2.98,
      "text": "现在很多 AI 已经很会识别模式，",
      "source": "funasr_sentence_with_term_fixes",
      "asr_text": "现在很多 AI 已经很会识别模式，"
    }
  ]
}

start/end 单位是秒。FunASR raw sentence_info.start/end 单位是毫秒，转换时除以 1000。

M1 Route B 执行规格

  1. 输入：一个含音轨的成片 MP4。
  2. Probe：ffprobe ... -of json input.mp4 > metadata.json，无音轨则 fail fast。
  3. 抽音频：ffmpeg -map 0:a:0 -ac 1 -ar 16000 project/audio/asr.wav。
  4. ASR：复用生产 funasr_transcribe.mjs 逻辑，在 director-console 新建本地 wrapper，输出 raw JSON 和 composer captions JSON。
  5. Transcript：M1 需新增 captions/raw → transcript.json 的确定性转换。
  6. Scene plan：当前无现成工具。M1 需新增 transcript → scene plan，最小版可按时长/标点规则分段，后续再接 LLM 辅助。
  7. Manifest：当前无 Route B scene plan → composer manifest 生成器。M1 需新增脚本，使用现有 component catalog、manifest validator 和 M0 compose API。
  8. Compose/render：后续使用现有 npm run validate:manifest、compose、lint、inspect、render；本次未跑渲染。
