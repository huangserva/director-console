# 设计稿视觉拆解（orchestrator 照 reference png 逐区拆，给实现用）

> 配套图：`director-packaging-console-ui-reference.png`。深色主题（近黑底 #0d0f12 级），金色高亮（#E8B23A 级），蓝/绿/紫做轨道色，科技/蜂巢感。整体 5 区：顶栏 / 左导航+面板 / 中画布 / 右 Inspector / 底（智能方案条 + 四轨时间线）。

## 1. 顶栏 TopBar
- 左：Logo「Director 包装工作台」（六边形图标 + 文字）
- 中：主流程 nav（横向）：导入素材 / 字幕识别 / 分析视频 / 风格包 / ✦智能包装(当前高亮) / ✓校验 / 导出设置
- 右：项目名「通用二次包装项目」 + `预览`(描边按钮) + `渲染`(金色实心按钮)

## 2. 左侧导航 + 面板
- 竖排图标 tab：项目 / 素材 / 场景 / **组件库(当前)** / 风格库 / 方案
- 组件库面板（当前）：
  - 搜索框「搜索包装组件」
  - 「组件分类 · 12 类」横向 chip 滚动：开场(当前金色) / 字幕 / 产品展示 / 讲解演示 / 对比 / …(可滚)
  - 「推荐卡片 | 可拖入时间线」
  - 卡片网格(2 列，每张=图标+标题+副标题)：
    产品亮点卡(卖点/功能/版本更新) · 章节标题卡(开场/转段/模块分隔) · 关键词字幕(逐句字幕/重点词强调) · 演示标注卡(截图/箭头/圈选/高亮) · 左右对比卡(方案/竞品/前后效果) · 画中画卡(人像/屏幕/反应小窗) · 数据证明卡(数字/引用/评价/证据) · 节奏歌词卡(MV/歌词/波形/节拍)

## 3. 中央画布 Canvas
- 工具条：✦光标 / ✋抓手 / 🔍缩放 / 「适配 46%」 | 右：16:9 横屏 / 安全区 / 网格 / 吸附(toggle)
- 画布(深色)：当前选中卡片「产品亮点卡片」——金色小标签「核心功能」+ 大标题「自动识别画面重点」+ 正文；四角金色选择手柄；右侧一个灰色 presenter 剪影(数字人位)；底部提示「这一段适合加入产品亮点和演示标注」

### 四轨时间线
- 左上：时间码 00:00:12:14 + ⏮▶⏭ + 吸附 / 渲染区间；右上「四类轨道：视频/音频/字幕/卡片」
- 标尺 00:00 → 00:40+
- 轨道(每条带 👁可见 / 🔒锁)：
  - **视频轨道**：主视频素材.mp4(蓝条，铺满)
  - **音频轨道**：原始音频.wav(绿条)
  - **字幕轨道**：字幕01 / 字幕02 / 字幕03(蓝描边块)
  - **卡片轨道**：开场标题卡 / 产品亮点卡(金) / 演示标注卡(紫) / 收尾引导卡(金)
- 播放头(竖黄线)在 ~12s

## 与现状 console-web 的关键差异（要补/要改）
1. 现状=左场景列表+props 编辑+只读时间条；目标=组件库面板 + 画布(卡片浮在 presenter 上) + Inspector(4 tab) + 四轨时间线 + 智能方案条。
2. 现状数据=扁平 scenes[]；目标=4 轨(视频/音频/字幕/卡片) + 卡片实例 schema(component-card.schema)。
3. 现状组件=composer catalog(StatsHero/StepCard…)；目标=12 类包装组件库 + 卡片实例字段(类型/内容/媒体/布局/字体/颜色/图形/动画/时间线/校验)。
4. 现状无 packaging-plan↔template 回环；目标=另存模板 + Codex 复用模板。
